• Prior to entering the diffusion sampling loop:

  – Decide if the target resolution is within the training distribution (i.e. results in token counts ≤ L_max) or out-of-distribution.
  – If the resolution is out-of-distribution and requires positional interpolation:
    ▸ Compute the scale factor based on the target resolution and training token length: 
         scale_factor = max(max(H_test, W_test) / L_train, 1.0).
    ▸ **Call a helper function (for example, set_extrapolation_embeddings) exactly once before entering the sampling iterations.** This function modifies the model’s positional embeddings using the chosen extrapolation method (one of: PI, EI, NTK, YaRN, VisionNTK, or VisionYaRN). It updates the positional embeddings or computes an adjusted version that will be used uniformly during the sampling loop.
  – This modification is performed once before the denoising iterations so that every call to model.forward during the sampling loop uses the correctly adjusted (extrapolated) positional embeddings.

• Diffusion Sampling Iterations:
  – With the extrapolated positional embeddings fixed, initialize a latent noise tensor (sampled from a Gaussian distribution) that matches the expected latent shape for the target resolution.
  – Set the model to evaluation mode and disable gradients.
  – Enter the iterative denoising loop for the fixed number of sampling steps (250 iterations):

  – Thus, the positional embedding modification is applied only once prior to the sampling loop and remains consistent throughout the iterations; there is no need to change it dynamically during each iteration.

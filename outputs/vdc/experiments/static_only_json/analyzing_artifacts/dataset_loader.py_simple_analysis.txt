  – Likely implementation: iterate through the dataset’s indices, decide (using a random draw) whether to inject noise, then update the label accordingly.
• b. Poisoned Sample Injection:
  – For each dataset, consult the configuration key dirty_sample_generation.poisoning.<DatasetName>.

    ∙ Randomly select the specified number of samples from the clean training samples (ensure they are drawn uniformly across classes).

• c. Hybrid Dirty Samples:

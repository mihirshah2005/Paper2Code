3. The Training Loop (in train())
  
• Outer Loop:  
  – Iterate over the total number of epochs (e.g., for epoch=0...epochs–1).  
  – At the start of each epoch, set the model to training mode (model.train()).

• For each epoch:
  
  1. Initialize accumulators for the loss and correct prediction counts to log training performance.  
  2. For every mini-batch extracted from the DataLoader:
     • Move the batch (inputs and labels) to the proper device (CPU/GPU).  
     • Perform a forward pass:  
       – Pass the inputs through the ResNet‑18 model to obtain predictions (logits).  
     • Compute the loss using the CrossEntropyLoss comparing logits to the true labels.  
     • Perform backpropagation:  
       – Zero the gradients (optimizer.zero_grad()).  
       – Call loss.backward() and optimizer.step() to update the model weights.
     • Accumulate the loss and compute the number of correct predictions for that batch.
  
  3. End of an Epoch:
     • Optionally, compute the training accuracy over all mini‑batches for the epoch using the accumulated correct predictions and total sample count.  
     • Log the epoch loss and accuracy.  
     • Step the learning rate scheduler so that it adjusts the learning rate if the current epoch matches any milestone (using scheduler.step()).
     • (Optional) Save model checkpoints if desired.
  
• Logging:  
  – For each epoch, record key metrics (loss, accuracy, and potentially learning rate).  
  – This logging helps track progress and troubleshoot if the model is not converging.

6. Pseudocode Summary

Below is a high-level pseudocode outline for Trainer:

------------------------------------------------------------------
class Trainer:
    def __init__(self, model, train_dataset, hyperparams, dataset_name):
        # 1. Store model and hyperparams
        self.model = model
        self.epochs = hyperparams["epochs"]
        self.lr = hyperparams["learning_rate"]
        self.batch_size = hyperparams["batch_size"][dataset_name]
        self.lr_schedule = hyperparams["lr_decay_schedule"][dataset_name]  # list with epochs
        
        # 2. Create DataLoader for train_dataset
        self.train_loader = DataLoader(train_dataset, batch_size=self.batch_size, shuffle=True, ...)
        
        # 3. Define optimizer (SGD) and loss function
        self.optimizer = torch.optim.SGD(self.model.parameters(), lr=self.lr)
        self.criterion = torch.nn.CrossEntropyLoss()
        
        # 4. Setup learning rate scheduler (MultiStepLR with milestones)
        self.scheduler = torch.optim.lr_scheduler.MultiStepLR(self.optimizer, milestones=[m["epoch"] for m in self.lr_schedule], gamma=0.1)
    
    def train(self):
        for epoch in range(self.epochs):
            self.model.train()
            running_loss, correct, total = 0.0, 0, 0
            
            for inputs, labels in self.train_loader:
                # Move batch to device (GPU/CPU)
                inputs, labels = inputs.to(device), labels.to(device)
                
                self.optimizer.zero_grad()
                outputs = self.model(inputs)
                loss = self.criterion(outputs, labels)
                loss.backward()
                self.optimizer.step()
                
                running_loss += loss.item() * inputs.size(0)
                predictions = outputs.argmax(dim=1)
                correct += (predictions == labels).sum().item()
                total += inputs.size(0)
            
            epoch_loss = running_loss / total
            epoch_acc = correct / total
            
            # Step the scheduler if epoch milestone reached
            self.scheduler.step()
            
            # Log the metrics (e.g., print or write to a log)
            print(f"Epoch {epoch+1}/{self.epochs}: Loss={epoch_loss:.4f}, Accuracy={epoch_acc:.4f}, LR={self.optimizer.param_groups[0]['lr']}")
            
            # Optionally: evaluate on a validation set / compute ASR if backdoor test set is provided
            
        # Optionally: Save final model checkpoint
        return self.model

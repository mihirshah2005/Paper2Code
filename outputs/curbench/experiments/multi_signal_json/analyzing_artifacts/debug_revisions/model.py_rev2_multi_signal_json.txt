 • Immediately check that the input domain is one of {“CV”, “NLP”, “Graph”}. If not, raise a ValueError with a message like "Unsupported domain: <domain>".

 • Confirm that required dataset-specific parameters (e.g., num_classes) are provided; if not, raise an exception alerting the user before proceeding.

2. Custom Learning Rate Scheduler  
─────────────────────────────  
• The Trainer must update the learning rate for every training step using the formula:  
  lr = d_model^(–0.5) * min(step_num^(–0.5), step_num * warmup_steps^(–1.5))
  – d_model is taken from the configuration.
  – step_num is the current global training step.
  – warmup_steps is from the config (4000).
• In every training iteration, the Trainer should:
  – Compute the current learning rate.
  – Loop through each parameter group in the optimizer and set its 'lr' field to the new value.
  – Log the updated learning rate for monitoring.

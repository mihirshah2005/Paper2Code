  • d_model as read from the configuration

 – Create an iterator over the train_loader and loop continuously until the global step count reaches total_steps.

 – Call loss.backward() to compute gradients.  
 – Call optimizer.step() to update model parameters.  
 – Zero the gradients afterward (optimizer.zero_grad()).

• Learning Rate Scheduling  
 – At every step, compute the current learning rate using the custom schedule.  
 – Update each parameter group's learning rate in the optimizer accordingly.
